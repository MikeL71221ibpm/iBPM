Emergency Process Recovery

Warning: Use these options only when processing has stalled or when directed by support. These actions cannot be undone.

Option 1: Force Stop Processing

Immediately stops all processing operations but keeps extracted data. Use this when processing appears stuck.

Option 2: Force Complete Processing

Marks the current process as completed regardless of its actual state. This can help when processing is near completion but stuck.

Option 3: Reset All Processing

Completely resets the processing state and clears all temporary data. Most severe option - use only when other options don't work.

Option 4: Boost Processing

Attempts to speed up processing by allocating more resources. Try this before more drastic measures.
